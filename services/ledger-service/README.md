# Ledger Service

The ledger-service is responsible for recording and exposing financial postings.

---

## Overview

This service persists immutable financial postings (ledger entries) generated by the transaction-service.

It acts as the **source of truth for financial history**, ensuring auditability and traceability.

---

## Running Locally

### Requirements

- Docker
- Java 17+ (or 21)
- Maven Wrapper (included)

### Start PostgreSQL

```bash
docker run --name ledger-postgres -e POSTGRES_USER=postgres -e POSTGRES_PASSWORD=postgres -e POSTGRES_DB=ledger_service -p 5432:5432 -d postgres:16
